# 背景音乐文件说明

## 文件要求

请将《好运来》音频文件放置在此目录下，命名为：**`haoyunlai.mp3`**

## 文件格式

- **格式**：MP3（推荐）
- **文件名**：`haoyunlai.mp3`（必须完全匹配）
- **文件大小**：建议不超过 5MB（为了加载速度）

## 使用方法

1. 将音频文件复制到此目录（`audio/haoyunlai.mp3`）
2. 刷新网页
3. 点击右下角的音乐按钮（🎵/🔇）开启/关闭背景音乐

## 功能特点

- ✅ 默认关闭状态
- ✅ 循环播放
- ✅ 音量控制（悬停音乐按钮显示音量滑块）
- ✅ 自动暂停（页面隐藏时）
- ✅ 设置持久化（刷新后保持状态）

## 注意事项

⚠️ **版权说明**：请确保您使用的音频文件拥有合法使用权，或使用无版权/已授权的音乐。

## 备用方案

如果不想使用本地文件，可以修改 `assets/js/background-music.js` 中的 `getMusicUrl()` 函数，使用在线音频资源URL。
